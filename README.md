# emerald_os

A structured database of the Emerald Tablets of Thoth for creative and AI-assisted purposes.

## Directory Structure

| Directory | Description | Files |
|-----------|-------------|-------|
| [`core/`](core/) | Core extraction data — entities, locations, forces, events, master tables | 65 files |
| [`visual/`](visual/) | Visual spectrum profiles for all entities, locations, and forces | 40 files |
| [`tablets/`](tablets/) | Tablet-by-tablet breakdown of the source material | 15 files |
| [`comparative/`](comparative/) | Cross-tradition and cross-mythology analysis | 15 files |
| [`cross_cuts/`](cross_cuts/) | Thematic slices across the database (by type, vibration, frequency, etc.) | 10 files |
| [`relationships/`](relationships/) | Network analysis — hierarchy maps, dependency webs, conflict matrices | 10 files |
| [`operations/`](operations/) | Practical protocols — invocations, keys, exit procedures, meditations | 12 files |
| [`indexes/`](indexes/) | Lookup tables — UUID index, glossary, anomaly log, master indexes | 10 files |
| [`metadata/`](metadata/) | Project documentation — passes, methodology, citations, license | 30 files |
| [`archive/`](archive/) | Original archive exports of the repository | 3 files |
